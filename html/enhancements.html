ENHANCEMENTS PAGE

This is the page where we show our enhancement features. We made 3 enhancements in total.

  1. Responsive
    2.

    - We have made every pages responsive by using media query. Certain elements will change when the screen width is less than 740px. In the link to enquire.html above, you can see the form changes from 2 collumns into one when the width is below 740px.

  2. Navigation's underline appears when hover
    3.

    - As you can see at our navigation bar, not only the line appears whenever the house hovers on an element, but the line actually appears from left to right. This gives customers a very pleasing and clean effect on our web page. We done this by using pseudo elements ::after to create the line. We then set position of the navigation item to relative in order to place the line in the appropriate position. Finally, we set original line's width as 0 then it can extend out to the wanted length when hover by using transition time.

  3. Color-changing button
    4.

    - We applied the same method to our button. We still use the pseudo element after to create a rectangle with the same size, positioning at the same place as the original button. We then have its width at 0 to begin with and transition out changing the background color whenever the button is hovered on.
